# Web Music Player

## Description

The Web Music Player is a simple web application that allows users to play music from a playlist. Users can play, pause, skip to the next or previous song, adjust the volume, and see the progress of the current song.

## Technologies Used

* HTML – for the structure of the music player
* CSS – for styling and layout
* JavaScript – for the player functionality and controls

## How the Files are Linked

The `index.html` file links to the `style.css` file for styling and the `script.js` file for functionality. JavaScript controls the music playback and connects the buttons and playlist to the audio player.

## Features

* Playlist display with song titles and artists
* Play and pause music
* Next and previous song controls
* Progress bar for song playback
* Volume control

## How to Run

1. Download or clone the project.
2. Add your MP3 files to the **music** folder.
3. Open `index.html` in a web browser.
